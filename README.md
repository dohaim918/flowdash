## 0. 프로젝트 개요 (간단 요약)
(※ 공통 과제이므로 1~2문단 이내로 간단히 작성)

본 프로젝트는 공통 요구사항을 기반으로 한  
칸반 형태의 태스크 관리 대시보드 구현 과제이다.

CRUD, 기간 필터, 통계, 테마 및 UX 요소를 포함하며  
팀 단위 협업을 통해 설계 및 구현을 진행했다.

## 3. 프로젝트 구조 및 아키텍처

### 3-1. 디렉터리 구조
```
flowdash/
├─ README.md
├─ index.html
├─ css/
│  ├─ reset.css             (resrt.css)
│  ├─ base.css              (style.css : 메인) 
│  ├─ components.css        (delete.css / modal_style.css : 재사용 )
│  ├─ theme.css             (색상변수:다크모드)
│  └─ responsive.css        (쿼리)
└─ js/
   ├─ main.js                (main.js : 이벤트리스너 ,,,)
   ├─ state.js               (deleteTodo.js : 삭제모달 / filter.js : 검색 파이프라인 ) 
   ├─ storage.js             (저장)
   ├─ utils/
   │  └─ date.js               
   └─ ui/                      (icons.js : svg / modal.js : 모달기능 / todoCard.js :)
      ├─ board.js              (dom)
      ├─ card.js               (top.js : 제목,내용,날짜)
      ├─ header.js             (로고버튼프로필)
      └─ stats.js              

       ※ todoCard.js : 카드생성 Ui + Dom + 모달역할 + 통계 
```

### 3-2. 모듈 책임 분리
- main.js : 이벤트 리스너 관리
- modal.js : Ui 생성과 모달 + 통계 관리
- deleteTodo.js : 앱의 상태인 Todo 리스트를 로컬 스토리지와 동기화하면서 관리
- icons.js : svg 관리
- filter.js : 검색 파이프라인 전담 관리 
- todoCard.js : Todo 앱에서 카드 생성, 모달, UI 상태, 통계 계산까지 관리
- top.js : 전역 UI 상태 관리 

### 3-3. 데이터 흐름
User Action  

→ Todo 데이터(todos) 상태 변경
→ LocalStorage에 상태 저장
→ 상태별 보드 다시 렌더링
→ 통계 UI 업데이트

## 6. 트러블슈팅 (Troubleshooting) > 팀원별 문제점 추가 <

### 6-1. (문제 제목)
- 증상: 드롭다운 커스텀 X / 정렬이 안됨 
- 원인: option으로 하면 커스텀이랑 정렬이 어렵다.
- 해결: ul il 활용하여 커스텀과 정렬 한번에 해결
- 회고: 앞으로 UI/UX 관련 기능은 초기 설계 단계에서 사용자 경험과 커스터마이징 가능성을 더 꼼꼼하게 고려해야한다는 점 

### 6-2. (문제 제목)
- 증상: 검색파이브라인 버튼 클릭 시 다른 이벤트 날아감
- 원인: 조건이 빠져있고 몇개는 주석처리
- 해결: 조건을 한곳에만 넣어두고 서로 상호작용 하는 부분에 안 넣음
- 회고: 주석처리라던가 조건이나 코드를 한두 번 더 체크하는 습관..

- ### 6-3. (문제 제목)

- 원인: 화면에 보이는 글자로 하면 첫번째 요소 텍스트만 가져옴

## 9. 결론

공통 과제를 거의 처음 경험하면서 협업의 방향성을 잡을 수 있었고, 그 과정에서 소통의 중요성을 확실히 체감했고,<br>
함께 작업할 때 서로를 배려하고 존중하는 태도가 협업의 완성도 높인다는 점 알게 되었으며,<br> 
요구사항과 코드 구조를 꼼꼼히 검토하고 모듈 책임을 분리 확실하게 해야겠다는 생각을 했다<br>
유지 보수뿐만 아니라 개인 개발에도 효율적이고 중요하다는 것을 느꼈는데 <br>
이러한 경험을 바탕으로 더 적극적으로 공통과제 or 협업에 임하고 싶ㄹ어졌다.
